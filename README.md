# brindasevak.github.io

Personal academic website of **Brinda Nishith Sevak**, PhD Candidate in Biomedical Engineering at the University of California, Davis.

🌐 **Live site:** [brinda3093.github.io](https://brinda3093.github.io)

## About

This site is built with [Jekyll](https://jekyllrb.com/) using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template, hosted on GitHub Pages.

## Structure

| Folder | Contents |
|---|---|
| `_pages/` | Main site pages (About, CV, etc.) |
| `_publications/` | Publication entries |
| `_talks/` | Conference talks & posters |
| `_teaching/` | Teaching experience |
| `_data/` | Navigation and other site data |
| `files/` | PDFs (CV, papers, etc.) |
| `images/` | Profile photo and other images |
